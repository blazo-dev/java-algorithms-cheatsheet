Stack / Queue
O(1) 
// STACK (LIFO)
Deque<Integer> stack = new ArrayDeque<>();
stack.push(val);      // agregar arriba
stack.pop();          // sacar arriba
stack.peek();         // ver sin sacar
stack.isEmpty();

// QUEUE (FIFO)
Queue<Integer> q = new LinkedList<>();
q.offer(val);         // agregar atrás
q.poll();             // sacar frente
q.peek();             // ver sin sacar
q.isEmpty();
Regla: Stack → push/pop/peek · Queue → offer/poll/peek

break / continue / label
Control flow

    if (found) break;
}

// continue — salta esta iteración
for (int i = 0; i < n; i++) {
    if (skip) continue;
}

// break en loop anidado con label
outer:

    for (int j = 0; j < m; j++) {

    }
}

O(n log n)

Comparator con lambda
Lambda 
// Ordenar strings por longitud
Arrays.sort(words, (a, b) -> a.length() - b.length());

// Ordenar 2D array por col 1
Arrays.sort(intervals, (a, b) -> a[1] - b[1]);

// Lista descendente
list.sort((a, b) -> b - a);

// Regla:
// a - b  →  ascendente
// b - a  →  descendente
Truco: negativo = a va primero · positivo = b va primero

Binary Search
O(log n) 
// Built-in (array debe estar ordenado)
int idx = Arrays.binarySearch(nums, target);
// Retorna índice si existe, negativo si no

// Implementación manual (más útil en entrevistas)
int left = 0, right = nums.length - 1;
while (left <= right) {
    int mid = left + (right - left) / 2;  // evita overflow
    if (nums[mid] == target)      return mid;
    else if (nums[mid] < target)  left  = mid + 1;
    else                          right = mid - 1;
}
return -1;
