Stack / Queue
O(1) 
// STACK (LIFO)
Deque<Integer> stack = new ArrayDeque<>();
stack.push(val);      // agregar arriba
stack.pop();          // sacar arriba
stack.peek();         // ver sin sacar
stack.isEmpty();

// QUEUE (FIFO)

q.poll();             // sacar frente
q.peek();             // ver sin sacar
q.isEmpty();
Regla: Stack → push/pop/peek · Queue → offer/poll/peek

Binary Search
O(log n) 
// Built-in (array debe estar ordenado)
int idx = Arrays.binarySearch(nums, target);
// Retorna índice si existe, negativo si no

// Implementación manual (más útil en entrevistas)
int left = 0, right = nums.length - 1;
while (left <= right) {
    int mid = left + (right - left) / 2;  // evita overflow
    if (nums[mid] == target)      return mid;
    else if (nums[mid] < target)  left  = mid + 1;
    else                          right = mid - 1;
}
return -1;

switch / switch expression
Switch 
// Switch clásico
switch (day) {
    case 1: result = "Lunes"; break;
    case 2: result = "Martes"; break;
    default: result = "Otro";
}

// Switch expression (Java 14+) — más limpio
String result = switch (day) {
    case 1 -> "Lunes";
    case 2 -> "Martes";
    default -> "Otro";
};
OJO: En el switch clásico siempre pon break o caerá al siguiente case.

Streams 
// Crear streams
list.stream()
Arrays.stream(arr)
Stream.of(1, 2, 3)

IntStream.rangeClosed(1, 10)    // 1..10

// Operaciones intermedias (lazy — no ejecutan hasta terminal)
.filter(x -> x > 0)             // filtrar
.map(x -> x * 2)                // transformar
.mapToInt(String::length)        // → IntStream
.flatMap(x -> x.stream())       // aplanar listas anidadas
.distinct()                      // eliminar duplicados
.sorted()                        // ordenar
.sorted((a,b) -> b - a)         // ordenar con comparator
.limit(n)                        // tomar solo n elementos
.skip(n)                         // saltar n elementos
.peek(x -> System.out.println(x))// debug sin modificar

// Operaciones terminales (ejecutan el pipeline)
.collect(Collectors.toList())    // → List
.collect(Collectors.toSet())     // → Set
.collect(Collectors.joining(", "))// → String
.toArray()                       // → array
.forEach(x -> use(x))           // iterar
.count()                         // contar elementos
.sum()                           // solo en IntStream
.min(Comparator.naturalOrder())  // → Optional
.max(Comparator.naturalOrder())  // → Optional
.findFirst()                     // → Optional primer elem
.anyMatch(x -> x > 0)          // → boolean
.allMatch(x -> x > 0)          // → boolean
.noneMatch(x -> x < 0)         // → boolean
.reduce(0, (a, b) -> a + b)     // → acumular en valor
